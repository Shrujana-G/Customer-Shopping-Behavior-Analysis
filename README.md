# Customer Behavior Data Analyst Portfolio Project

## 📊 Project Overview

This project demonstrates an end-to-end **Data Analytics workflow** by transforming raw customer shopping data into meaningful business insights. It covers the complete analytics process, including data cleaning, exploration, SQL analysis, dashboard creation, and business recommendations.

The project simulates a real-world analyst workflow used in organizations to understand customer behavior, purchasing patterns, and business performance.

## 🎯 Objectives

* Clean and preprocess raw customer data using Python
* Perform exploratory data analysis (EDA) to identify trends and patterns
* Analyze customer behavior using SQL queries
* Build an interactive Power BI dashboard
* Generate business insights and recommendations from data

## 🛠️ Tech Stack

* **Python**

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn

* **SQL**

  * MySQL / PostgreSQL / SQL Server

* **Power BI**

  * Data modeling
  * Interactive dashboards
  * Data visualization

## 🔄 Project Workflow

### 1. Data Preparation & EDA (Python)

* Import raw customer dataset
* Explore data structure and statistics
* Handle missing values and inconsistencies
* Perform data cleaning and transformation
* Identify customer trends and patterns

### 2. Database Analysis (SQL)

* Load cleaned data into SQL database
* Create tables and structure data
* Write SQL queries to analyze:

  * Customer segments
  * Purchase behavior
  * Loyalty patterns
  * Revenue trends
  * Product performance

### 3. Dashboard Development (Power BI)

Created an interactive dashboard to visualize:

* Customer demographics
* Shopping trends
* Sales insights
* Customer preferences
* Key business metrics

### 4. Reporting & Insights

* Summarize analytical findings
* Identify business opportunities
* Provide data-driven recommendations

## 🚀 How to Run the Project

### Clone Repository

```bash
git clone <repository-url>
cd Customer-Behavior-Data-Analysis
```

### Run Python Notebook

Open:

```
Customer_Shopping_Behavior_Analysis.ipynb
```

Execute steps for:

* Data loading
* Cleaning
* EDA
* Database connection
* Data upload

### Run SQL Analysis

Open:

```
customer_behavior_sql_queries.sql
```

Execute queries in your SQL database environment.

### Open Power BI Dashboard

Open:

```
customer_behavior_dashboard.pbix
```

Connect with the database and explore insights.
